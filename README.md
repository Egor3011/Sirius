# Telegram бот MaryTravel

Бот для туристического агентства MaryTravel с информацией о турах на Байкал, Дагестан, Алтай и Стамбул.

## Запуск

### Вариант 1: Через Docker (рекомендуется)
```bash
./start_bot.sh
```

### Вариант 2: Ручной запуск через Docker
```bash
# Сборка образа
docker build -t telegram-bot .

# Запуск контейнера
docker run -d --name telegram-bot-container telegram-bot
```

### Вариант 3: Прямой запуск (если Docker недоступен)
```bash
pip install -r requirements.txt
python3 main.py
```

## Команды бота
- `/start` - Главное меню с кнопками
- `/schedule` - Расписание туров
- `/regions` - Выбор региона тура
- `/faq` - Часто задаваемые вопросы
- `/id` - ID сообщения (только для разработчика)

## Структура проекта
- `main.py` - Основной код бота
- `config.py` - Настройки и константы
- `exel.py` - Работа с Excel файлом для статистики
- `requirements.txt` - Зависимости Python
- `Dockerfile` - Конфигурация Docker

## Проблемы и решения

### Ошибка 409 (Conflict)
Если возникает ошибка "terminated by other getUpdates request", значит где-то еще запущен бот с тем же токеном:
1. Остановите все другие экземпляры бота
2. Проверьте, что токен в `config.py` уникален
3. Перезапустите бота

### Сетевые проблемы
Бот использует Docker для стабильной работы с сетью. Если Docker недоступен, запустите напрямую с правами администратора.
